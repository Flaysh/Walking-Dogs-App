## Run Locally

Clone the project

```bash
  git clone https://github.com/Flaysh/Walking-Dogs-App
```

Go to the project directory

```bash
  cd walking-dogs-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
